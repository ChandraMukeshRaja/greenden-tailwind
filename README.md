# greenden-tailwind
GreenDen

A simple plant purchase website built using Tailwind CSS and Grid.

Features

- Responsive design for mobile and desktop screens
- Grid layout for showcasing plant cards
- Basic plant information and pricing

Technologies Used

- HTML
- Tailwind CSS
- Grid layout
